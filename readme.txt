An advanced, fully responsive table widget for Elementor with sticky headers, custom content (images, buttons), and flexbox ordering.

== Description ==

Advance Table For Elementor is the ultimate solution for creating complex, responsive, and beautiful tables directly inside Elementor. Unlike standard HTML tables, this widget gives you full control over styling, layout, and content.

Whether you need a simple data table, a complex pricing table, or a feature comparison chart, this plugin handles it all with zero coding.

ðŸš€ Key Features

Fully Responsive: Automatic horizontal scrolling on mobile devices to prevent layout breakage. You can set a custom minimum width for mobile views.

Advanced Content Support: Add Images, Logos, Titles, Descriptions, and Buttons directly inside any table cell.

Flexbox Ordering: Reorder elements (Image, Title, Content, Button) inside a cell using a simple number system. No CSS required!

Sticky Header: Keep your table header visible while scrolling through long data lists.

Merge Cells: Full support for Rowspan (Vertical Merge) and Colspan (Horizontal Merge).

Styling Controls: Extensive styling options for Table Head (TH), Table Body (TD), Borders, Typography, Colors, and Padding.

Custom Column Widths: Define precise column widths using percentages.

Data Reports: Show large datasets with sticky headers for better readability.

Schedules: Create event schedules with merged cells for time slots.

== Installation ==

Upload the advance-table-for-elementor folder to the /wp-content/plugins/ directory.

Activate the plugin through the 'Plugins' menu in WordPress.

Make sure Elementor is installed and active.

Open any page with Elementor Editor.

Search for the "Advance Table" widget and drag it to your section.
